# Inventory Management System

Консольна програма для управління складськими запасами з можливістю додавання, видалення та перегляду товарів.

## 📦 Огляд проекту

Цей Java-додаток дозволяє керувати інвентарем товарів через консольний інтерфейс. Система забезпечує базові операції з управління запасами.

## ✨ Функціонал

### Основні можливості
- **📥 Додавання товарів** - додавання нових позицій до інвентарю
- **🗑️ Видалення товарів** - видалення товарів за назвою
- **👀 Перегляд інвентарю** - відображення всіх наявних товарів
- **❌ Валідація даних** - обробка помилок вводу користувача

### Інформація про товар
Кожен товар містить:
- **Назва** (name) - унікальний ідентифікатор товару
- **Кількість** (quantity) - кількість одиниць на складі
- **Ціна** (price) - вартість одиниці товару
